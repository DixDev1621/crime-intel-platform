# Crime Intelligence Platform

An AI-powered Crime Intelligence Platform designed to assist law enforcement agencies with crime management, investigation, and predictive analytics. The platform centralizes crime records, visualizes criminal networks, and leverages machine learning to identify crime hotspots for proactive policing.

---

## 🚀 Features

- Secure User Authentication
- FIR Management System
- Criminal & Victim Database
- AI-Based Crime Hotspot Prediction
- Criminal Network Visualization
- Investigation Support Dashboard
- Real-Time Crime Analytics
- Interactive Reports and Insights

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
